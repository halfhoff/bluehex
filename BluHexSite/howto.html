How to install BLUHEX

  1. Open Google Chrome Browser window

  2. Go to the Chrome Web Store

  3. Find and select the extension "Bluhex"

  4. Click the + Add to Chrome button

  5. Follow the prompt to add extension

  6. A Bluhex icon will appear on your tool bar

Manage BLUHEX

To control the settings of Bluhex left click on the icon in the top right of Chrome

The navigation will display the features:
On/Off Switch
Password
Time of Day
Word Filter
Timer
GPS

To remove the extension right click the icon and select remove
